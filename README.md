# Walk His Journey

A lightweight React + Vite prototype for tracking the journeys of Jesus with manual mile logging, prayer walk prompts, and a simple journal stored in localStorage.

## Getting started

```bash
npm install
npm run dev
```

Run the production build with:

```bash
npm run build
```

## Tech stack
- React with React Router
- Vite bundler
- Tailwind CSS
- LocalStorage for progress and journal persistence
